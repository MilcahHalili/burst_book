## Getting Started

**[Use the App] (https://burstbook.herokuapp.com/)**

### *How To Use*

Create a custom flip book by doing the following:

- SIGN UP or LOG IN to get started.
- Click on CREATE A BOOK.
- Upload burst photos.
- Submit delivery and payment info.
